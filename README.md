# PDS2-Q2

Execute the `run.sh` file to generate the binary files inside `dist` folder and get packages.

For OSX:
```shell
sh ./run.sh
```

For Windows:
```shell
./run.sh
```

Execute the `bootstrap.sh` file to generate the `main.exe` file.

For OSX:
```shell
sh ./bootstrap.sh
```
For Windows:
```shell
./bootstrap.sh
```
